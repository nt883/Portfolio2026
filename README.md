# ✦ Ntando Lutendo — Portfolio

> **Live → [nt883.github.io/Portfolio2026](https://nt883.github.io/Portfolio2026/)**

Personal developer portfolio built from scratch — no frameworks, no templates, no shortcuts. Just clean vanilla HTML, CSS, and JavaScript doing exactly what it needs to do.

---

## ⚡ Features

- ✦ **Typewriter hero** — cycles through roles on load
- 🌐 **Particle canvas** — animated dot cluster with live connecting lines, built with the Canvas API
- 🎞️ **Scroll reveal animations** — elements fade in as you scroll using IntersectionObserver
- 💫 **Glow effects** — subtle cyan aura on hover interactions  
- 📱 **Fully responsive** — looks sharp on mobile and desktop
- 🚫 **Zero dependencies** — no npm, no build step, open and it works

---

## 🗂️ Structure

```
Portfolio2026/
├── index.html      # Structure & content
├── styles.css      # Styling, animations, responsive layout
└── script.js       # Typewriter, particle canvas, scroll reveal
```

## 🚀 Run locally

```bash
git clone https://github.com/nt883/Portfolio2026.git
cd Portfolio2026
open index.html
```

Or use **Live Server** in VS Code for hot reload.
